## What is WEBP?

WEBP is a modern image format developed by Google. It produces significantly smaller file sizes than JPG and PNG at equivalent quality, which is why websites use it heavily for performance. However, WEBP isn't always supported by older apps, image editors, and document tools — which is why converting to JPG is often necessary.

## Why convert WEBP to JPG?

- **App compatibility** — many image editors, Word, and older tools don't open WEBP
- **Printing** — most print services require JPG or PNG
- **Email** — some clients don't render WEBP inline
- **Downloaded images** — sites serve WEBP but you need JPG for editing
- **Social sharing** — some platforms still prefer JPG
